# Student Management System

A full-stack Student Management System built while learning modern web development.

## Features

* Add Students
* View Students
* Edit Students
* Delete Students
* Search Students
* Sort Students by Name
* Sort Students by Marks
* Dashboard Statistics

  * Total Students
  * Average Marks
  * Highest Marks
  * Lowest Marks
  * Passed Students
  * Failed Students

## Technologies Used

### Frontend

* React
* Vite
* Axios
* CSS

### Backend

* Node.js
* Express.js

## API Endpoints

### Get All Students

GET `/students`

### Get Student By ID

GET `/students/:id`

### Add Student

POST `/students`

### Update Student

PUT `/students/:id`

### Delete Student

DELETE `/students/:id`
